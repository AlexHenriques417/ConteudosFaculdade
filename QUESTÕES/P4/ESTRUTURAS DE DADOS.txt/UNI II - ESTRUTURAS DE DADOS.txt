P4 - ESTRUTURAS DE DADOS 

QUESTÕES

1. As estruturas de dados lineares são caracterizadas por:
a) Não possuírem ordem definida
b) Permitir inserções e remoções em qualquer posição
c) Serem organizadas de forma sequencial, com predecessor e sucessor
d) Terem nós conectados de forma hierárquica

Resposta : Letra C

2. Em uma lista encadeada, cada nó:
a) Guarda apenas o conteúdo
b) Guarda o conteúdo e o ponteiro para o próximo nó
c) Guarda apenas o ponteiro do nó anterior
d) Aponta sempre para o primeiro nó da lista

Resposta : Letra B

3. Nas listas duplamente encadeadas, cada nó:
a) Aponta apenas para o próximo nó
b) Aponta apenas para o nó anterior
c) Guarda o conteúdo, o nó predecessor e o sucessor
d) Está sempre ligado ao nó raiz

Resposta : Letra C

4. Em uma fila, os elementos são organizados segundo a política:
a) LIFO – Last In, First Out
b) FIFO – First In, First Out
c) FILO – First In, Last Out
d) LILO – Last In, Last Out

Resposta : Letra B

5. A pilha é uma estrutura de dados onde:
a) O primeiro elemento a entrar é o primeiro a sair
b) O último elemento a entrar é o último a sair
c) O último elemento a entrar é o primeiro a sair
d) Todos os elementos são acessados simultaneamente

Resposta : Letra C

6. O deque (Double Ended Queue) permite:
a) Inserção apenas no início
b) Inserção e remoção em ambos os extremos
c) Acesso apenas ao último elemento
d) Somente leitura dos elementos

Resposta : Letra B

7. Em uma árvore binária, os nós:
a) São organizados de forma linear
b) Sempre possuem dois pais
c) São conectados por arestas e formam uma hierarquia
d) Estão dispostos em formato circular

Resposta : Letra C

8. Um nó folha em uma árvore é:
a) Um nó que possui apenas um filho
b) Um nó que não possui filhos
c) O primeiro nó da árvore
d) Um nó com grau maior que zero

Resposta : Letra B

9. A especialização é um tipo de relacionamento que indica:
a) “É parte de”
b) “É feito de”
c) “É um” ou “É uma”
d) “Depende de”

Resposta : Letra C

10. Na composição, podemos afirmar que:
a) As partes existem independentemente do todo
b) O todo e as partes são independentes
c) A parte não existe sem o todo
d) Cada parte tem seu próprio ciclo de vida

Resposta : Letra C

11. A hierarquia de dependência serve para:
a) Determinar o tamanho de uma árvore binária
b) Identificar quais entidades são subordinadas a outras
c) Calcular a altura de uma árvore
d) Organizar a ordem das operações matemáticas

Resposta : Letra B

12. O que é uma floresta em estrutura de dados?
a) Uma árvore com muitos galhos
b) Conjunto de árvores binárias balanceadas
c) Conjunto de zero ou mais árvores disjuntas, sem pontos em comum
d) Uma árvore com grau máximo igual a 3

Resposta : Letra C

13. Em uma árvore binária de busca (ABB), os elementos armazenados à esquerda de um nó pai são:
a) Maiores que o nó pai
b) Iguais ao nó pai
c) Menores que o nó pai
d) Aleatórios

Resposta : Letra C

14. A altura de uma árvore é definida como:
a) A distância da raiz até o primeiro nó
b) O número total de nós
c) O maior nível alcançado por alguma das folhas
d) O caminho mais curto entre dois nós

Resposta : Letra C

15. Uma árvore está balanceada quando:
a) Todos os nós possuem dois filhos
b) Todas as folhas estão à mesma distância da raiz
c) O número de folhas é igual ao número de níveis
d) A raiz tem grau máximo 2

Resposta : Letra B

16. A altura (H) e o número total de nós (N) de uma árvore binária completa estão relacionados pela fórmula:
a) N = H² + 1
b) N = 2H - 1
c) N = 2^H - 1
d) N = H × 2

Resposta : Letra C

17. O nível de um nó em uma árvore é:
a) A distância do nó até a raiz
b) A quantidade de filhos que o nó possui
c) Sempre o mesmo que sua profundidade
d) O número de folhas que ele gera

Resposta : Letra A

18. O grau de uma árvore representa:
a) O número total de níveis
b) O maior grau (número de filhos) de um nó dessa árvore
c) O número de folhas existentes
d) A profundidade máxima da raiz

Resposta : Letra B

19. Uma árvore degenerada é aquela em que:
a) Todos os nós têm dois filhos
b) O número de níveis é igual ao número de nós
c) As folhas estão balanceadas
d) Não possui raiz

Resposta : Letra B

20. A quantidade de nós em um determinado nível (H) de uma árvore binária é dada por:
a) 2^(H-1)
b) H²
c) H × 2
d) 2^H - 1

Resposta : Letra A

21. A estrutura abaixo representa corretamente um nó de uma árvore binária:

estrutura No {
   int valor;
   No esquerda;
   No direita;
}

O que essa estrutura permite fazer?
A. Criar uma lista linear
B. Criar uma fila encadeada
C. Criar um nó e gerar a árvore
D. Criar apenas a raiz sem filhos

Resposta : Letra C

22. Se a árvore estiver vazia, o que deve ser feito ao inserir o primeiro elemento?
A. Inserir o valor na direita da raiz
B. Criar a árvore e o primeiro nó
C. Aguardar outro valor ser inserido
D. Pular a inserção

Resposta : Letra B

23. Quando o valor a ser inserido for menor que o valor do nó atual, ele deve ser inserido:
A. Na direita do nó atual
B. Na esquerda do nó atual
C. Na raiz da árvore
D. No último nó da direita

Resposta : Letra B

24. No percurso em ordem (em-ordem), a sequência correta de visitação dos nós é:
A. Raiz, Esquerda, Direita
B. Direita, Raiz, Esquerda
C. Esquerda, Raiz, Direita
D. Esquerda, Direita, Raiz

Resposta : Letra C

25. O percurso pré-ordem segue a seguinte sequência:
A. Raiz, Esquerda, Direita
B. Esquerda, Direita, Raiz
C. Esquerda, Raiz, Direita
D. Direita, Esquerda, Raiz

Resposta : Letra A

26. O percurso pós-ordem visita os nós na seguinte ordem:
A. Raiz, Direita, Esquerda
B. Esquerda, Direita, Raiz
C. Direita, Esquerda, Raiz
D. Esquerda, Raiz, Direita

Resposta : Letra B

27. O percurso em largura (busca em nível) tem como principal característica:
A. Utiliza pilha
B. Percorre por níveis, da esquerda para a direita
C. Utiliza recursão
D. Visita primeiro as folhas da árvore

Resposta : Letra B

28. Para realizar a busca em nível, é necessário usar:
A. Pilha
B. Fila
C. Vetor
D. Lista circular

Resposta : Letra B

29. Ao deletar um nó folha, o que acontece?
A. É necessário reorganizar toda a árvore
B. O nó é apenas removido, sem realocação
C. O nó da direita assume o lugar
D. A raiz é substituída

Resposta : Letra B

30. Quando se deleta um nó com dois filhos, o procedimento correto é:
A. Substituir o nó por qualquer outro valor
B. Deletar a árvore inteira
C. Escolher um lado e usar o maior valor desse lado para substituir o nó removido
D. Apenas ignorar a remoção

Resposta : Letra C
