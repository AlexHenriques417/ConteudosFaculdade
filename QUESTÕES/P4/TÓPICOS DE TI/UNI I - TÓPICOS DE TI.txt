P4 - TÓPICOS DE TI 

QUESTÕES

1. Sobre o TypeScript, é correto afirmar que:
a) É uma linguagem que substitui totalmente o JavaScript.
b) Não oferece nenhum recurso adicional em relação ao JavaScript.
c) É uma linguagem que compila para JavaScript, oferecendo tipagem estática. 
d) É usada apenas para desenvolvimento de back-end.

Resposta : Letra C

2. Uma das vantagens do TypeScript em relação ao JavaScript é:
a) Aumentar a chance de erros em tempo de execução.
b) Evitar erros de tipo antes da execução do código. 
c) Ser menos legível que o JavaScript.
d) Remover a necessidade de compilar para rodar no navegador.

Resposta : Letra B 

3. Na pirâmide de linguagens/tecnologias apresentadas, a ordem correta é:
a) JS → ES6 → TYPE 
b) TYPE → JS → ES6
c) ES6 → TYPE → JS
d) JS → TYPE → ES6

Resposta : Letra A

4. Qual das opções apresenta corretamente um tipo primitivo em TypeScript?
a) integer
b) string 
c) float
d) char

Resposta : Letra B

5. Para representar um tipo que pode ser string ou number, usamos em TypeScript:
a) union type 
b) any
c) array
d) tuple

Resposta : Letra A

6. Qual dos símbolos pode ser usado para declarar uma string em TypeScript/JavaScript?
a) #texto#
b) 'texto' 
c) <texto>
d) (texto)

Resposta : Letra B

7. O operador ternário é utilizado para:
a) Repetir instruções diversas vezes.
b) Definir variáveis globais.
c) Fazer uma verificação condicional em uma única linha. 
d) Declarar funções assíncronas.

Resposta : Letra C

8. Qual loop garante que o bloco de código seja executado pelo menos uma vez?
a) for
b) while
c) do while 
d) for…of

Resposta : Letra C

9. No modelo Cliente-Servidor, a sequência correta é:
a) Cliente → HTTP Response → Servidor
b) Cliente → HTTP Request → Servidor 
c) Servidor → DNS → Cliente
d) Cliente → Banco de Dados → Servidor

Resposta : Letra B

10. Em operações CRUD, a letra “U” representa:
a) Upload
b) Update 
c) Unlock
d) Undo

Resposta : Letra B
