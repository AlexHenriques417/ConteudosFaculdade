- QUANDO UM USUÁRIO VISITA UMA PÁGINA WEB
SERVIDOR RETRONA UM ARQUIVO HTML PARA O BROWSER
ELE LÊ O HTML
E CONSTROI O DOM 

- DOM
É UM PADRÃO DE RECUPERAÇÃO, ALTERAÇÃO, ADIÇÃO OU REMOÇÃO
É CRUD
O DOCUMENTO INTEIRO É UM NÓ
CADA ATRIBUTO DO HTML É UM ATRIBUTO DE UM NÓ

- JAVASCRIPT E DOM
PODE UTILIZAR OS MÉTODOS PARA EXECUTAR EVENTOS
MANIPULAÇÃO COMO SELENCIONANDO, ATUALIZANDO E EXECLUINDO
TUDO NA INTERFACE DO USUÁRIO
O DOM PERMITE ALTERAR SEU ESTILO E CONTEÚDO

- EVENTOS
SÃO AÇÕES - PODEM SER DETECTADAS EM JAVASCRIPT
PODEMOS CRIAR PÁGINAS DINÂMICAS
SÃO ESPECIFICADOS E DIFINIDOS EM TAGS HTML
ONCHANGE, ONCLICK, ONMOUSEOVER, ONMOUSEOVER,ONKEYDOWN

- IMPORTANTE
HTML SEMPRE É FIXO E PERMANECE O ORIGINAL QUANDO VOCÊ CRIOU
DOM SEMPRE ESTÁ SE MODIFICANDO

- REACT
