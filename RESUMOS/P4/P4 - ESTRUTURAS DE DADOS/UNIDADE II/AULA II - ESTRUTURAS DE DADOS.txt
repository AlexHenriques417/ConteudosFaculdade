AULA II - ESTRUTURAS DE DADOS - UNIDADE II

- HIERARQUIA DE DEPENDÊNCIA
IDENTIFICA QUAIS ENTIDADES SÃO SUBORDINADAS A OUTRAS
ELEMENTOS QUE PODEM SER ENCONTRADOS DENTRO DOS OUTROS
ORGANIZAÇÃO DE ARQUIVOS - DIRETÓRIOS
QUAL PASTA É ACHADA DENTRO DE QUAL PASTA

- APLICAÇÕES DDE ÁRVORES
2 + 3 P COMPILADOR CONVERTE PARA + 2 3 
PRIMEIRO VEM A OPERAÇÃO DEPOIS VEM OS OPERANDOS 
2 + 3 * 4 FICA + 2 * 3 4 
USANDO ESSA NOTAÇÃO A EXPRESSÃO ENTREGA OS DADOS NA SEQUÊNCIA QUE DEVE SER COLOCADO NA ÁRVORES

- IMPORTANTE
A ÁRVORE É PERCORRIDA DE CIMA PARA BAIXO, E DA ESQUERDA PARA A DIREITA

- ORDENAÇÃO E BUSCA DE VALORES
ÁRVORES SÃ BASTANTE USADAS NA COMPUTAÇÃO PARA ORDENAR VALORES E BUSCAR TAISS VALORES PRESENTES NELA

- EXEMPLO
SISTEMAS OPERACIONAIS
BANCO DE DADOS 
COMPUTAÇÃO GRÁFICA 
USAM ÁRVORES

- FLORESTA
CONJUNTO DE ZERO OU MAIS ÁRVORES DISJUNTAS
É PRECISO QUE TODAS ESTEJAM ISOLADAS UMAS DAS OUTRAS
SEM QUALQUER PONTO EM COMUM

- NÍVEL
É A DISTÂNCIA DO NÓ ATÉ A RAIZ
A RAIZ SEMPRE ESTÁ NO NÍVEL 1
O NÍVEL É SEMPRE FIXO

- ALTURA
A ALTURA DE UMA ÁRVORE É O MAIOR NÍVEL ALCANÇADO POR ALGUMA DAS FOLHAS DA ÁRVORE
MAIOR CAMINHO DENTRO DE UMA ÁRVORE, COMEÇANDO PELA RAIZ

- CAMINHO E COMPRIMENTO
CAMINHO - PERCURSO REALIZADO PARA ATINGIR ALGUM NÓ PARTINDO DE OUTRO NÓ
COMPRIMENTO - QUANTIDADE DE NÓS NO PERCORRER DO CAMINHO

- EXEMPLO
CAMINHO DE B ATÉ K - 3 ( B, F, K)
COMPRIMENTO( TAMANHO ) DE B ATÉ K - 3 NÓS ( 2 ARESTAS )

- PROFUNDIDADE
DISTÂNCIA DE UM NÓ ATÉ A RAIZ DA ÁRVORE
A QUANTIDADE DE NÓS PELOS QUAIS SE DEVE PASSAR PRA CHEGAR DE UM NÓ ATÉ A RAIZ
ESTANDO NA RAIZ, ELE PRECISA SÓ DE UM NÓ MESMO PARA CHEGAR NA RAIZ

- ALTURA DE UM NÓ
QUANTIDADE DE NÓS EXISTENTES ENTRE ESTE NÓ E A FOLHA MAIS PROFUNDA DELE NA ÁRVORE CONSIDERADA

- LEMBRETE
AO CALCULAR A ALTURA DE UM NÓ É PRECISO SABER QUAL ÁRVORE ESTÁ SENDO CONSIDERADA

- ÁRVORE BALANCEADA
A ÁRVORE ESTÁ BALANCEADA QUANDO TODAS AS SUAS FOLHAS FICAM A MESMA DISTÂNCIA DA RAIZ

- GRAU DE UMA ÁRVORE
MAIOR GRAU ALCANÇADO POR ALGUM DOS NÓS DESSA ÁRVORE

- DEFINIÇÃO
UMA ÁRVORE É UM CONJUNTO FINITO T DE ZERO OU MAIS NÓS TAL QUE :

1. QUANDO O NÚMERO DE NÓS É DIFERENTE DE ZERO
2. QUANDO O NÚMERO DE NÓS É IGUAL A ZERO ( A ÁRVORE ESTÁ VAZIA )

- ÁRVORES BINÁRIAS
O GRAU DA ÁRVORE É 2
QUALQUER NÓ SÓ PODE TER 0, 1 OU NO MÁXIMO 2 SUCESSORES
AS SUBÁRVORES PODEM TER ESQUERA E DIREITA
ÁRVORE BINÁRIA COMPLETA - É AQUELA EM QUE TODOS OS NÓS POSSUEM EXATAMENTE 2 FILHOS, EXCETO AS FOLHAS

- ÁRVORE BINÁRIA DE BUSCA ( ABB )
CADA NÓ DA ÁRVORE ARMAZENA UM ELEMENTO
ELEMENTOSS ARMAZENADOS NA ESQUERDA SEMPRE SERÃO MENORES QUE O NÓ PAIS
ELEMENTOS ARMAZENADOS NA DIREITA SEMPRE SERÃO MAIORES QUE OS NÓS PAIS

- TOTAL DE NÓS DE UMA ÁRVORE BINÁRIA
SÓ PODE SER CALCULADO EM ÁRVORES BINÁRIAS COMPLETAS OU CHEIAS
A QUANTIDADE DE NÓS PODE SER CALCULADA PELA EQUAÇÃO = 2^H - 1
EM QUE H É ALTURA

- EXEMPLO 
SUPONDO UMA ÁRVORE DE ALTURA 3, ESTA TERIA 7 NÓS 
2^3 - 1 = 8 - 1 = 7

- TOTAL DE NÓS POR NÍVEL
A QUANTIDADE DE NÓS QUE EXISTE EM UM CERTO NÍVEL DE UMA ÁRVORE
EQUAÇÃO - 2^H-1, EM QUE H É O NÍVEL CONSIDERADO

- EXEMPLO 
NÍVEL 1 - RAIZ
2^1-1 = 2^0 = 1

POR ESSE MOTIVO, A RAIZ NÃO PODE SER CONSIDERADA NÍVEL ZERO E ALTURA ZERO

- SEQUÊNCIA NÚMERICA
ESSE TIPO DE ÁRVORE É CHAMADA DE ÁRVORE DEGENERADA, POIS O NÚMERO DE NÍVEIA DA ÁRVORE É IDÊNTICO AO NÚMERO DE NÓS QUE ELA POSSUI
